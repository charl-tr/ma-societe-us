09 — Diagnostic

Gaps critiques du site actuel

🔴
Caps lock omniprésent sur tous les titres Ton "direct response" années 2010, incompatible avec un positionnement de cabinet premium. Signal de site vieillissant.
🔴
1 seul témoignage (Antoine R.) sans photo ni prénom complet Insuffisant pour justifier $997–$1597 et lever les objections de confiance. 500 clients → 1 témoignage visible.
🔴
Aucun prix sur la page de vente principale (/creer-llc-offre/) Le visiteur qui arrive sur "Notre offre" doit naviguer vers une autre page pour voir le tarif. Friction de conversion directe.
🔴
Contact sans formulaire, email, ni téléphone publics Seul moyen de contact = Calendly. Un prospect qui ne veut pas prendre RDV immédiatement n'a aucune autre option → perte de lead.
🟠
Blog à l'arrêt depuis mars 2024 (0 article en 2025–2026) Perte de signal SEO, signal de site non maintenu. Pourtant le blog génère du trafic froid qualifié.
🟠
Aucune page "À propos" / équipe dans la nav Christophe est le principal signal de confiance (fondateur depuis 2014, expert nommé par les clients). Il est invisible dans la navigation principale.
🟠
Expérience fragmentée sur 3 domaines ma-societe-us.com / formations.ma-societe-us.com / app.ma-societe-us.com. Le visiteur change d'URL pour acheter, se former, ou accéder à son espace — sans cohérence visuelle.
🟠
Visa E2 = service orphelin dans le funnel Audience, intent, ticket, processus de vente totalement distincts du cœur LLC. Mélanger les deux dilue le message principal.
🟡
Section "offshore" en position 3 sur la homepage Introduit le doute ("c'est illégal de mettre de l'argent à l'étranger") avant de le lever. Réduire la friction cognitive : remonter la section avantages, reléger la définition offshore.
🟡
CTA "Formation gratuite" → renvoie vers un écran de login Un prospect non-client qui clique sur "Formation" arrive sur un formulaire de connexion. Friction maximale pour un lead froid.
