# Analyse des Tunnels de Conversion — 3 Sites

## 1. MA-SOCIETE-US.COM (site actuel)

### Sitemap complet
```
/                           Homepage
├── /creer-llc-offre/       Notre offre (détail)
├── /ouvrir-compte-bancaire-usa-solution/   Banque
├── /monter-societe-usa-pourquoi/           FAQ
├── /llc-usa/               Création LLC (page pilier + pricing $997/$1597)
├── /declaration-llc/       Déclaration LLC
├── /monter-societe-etats-unis-tarifs/      Prestations additionnelles
├── /visa-e2/               Immigration US (Visa E2)
├── /blog/                  Blog (articles depuis 2021)
│   ├── /declarations-fiscales-usa/
│   ├── /operating-agreement/
│   ├── /dissolution-llc/
│   └── /boi-fincen/
├── /creer-entreprise-etats-unis-contact/   Contact
├── /legalite-societe-americaine/           Légalité
└── EXTERNES :
    ├── calendly.com/ypls/decouverte-site           Calendly (RDV)
    ├── formations.ma-societe-us.com/...            Formation gratuite
    └── app.ma-societe-us.com                       Portail client
```

### Points de conversion (CTAs)
```
HOMEPAGE :
  CTA 1 → Calendly (entretien 15 min)          [PRIMAIRE]
  CTA 2 → Formation gratuite (lead magnet)      [SECONDAIRE]
  CTA 3 → /creer-llc-offre/ (notre offre)       [EXPLORATION]
  CTA 4 → /creer-entreprise-etats-unis-contact/ [CONTACT]
  CTA 5 → /monter-societe-usa-pourquoi/ (FAQ)   [RASSURANCE]

/llc-usa/ (page pilier) :
  CTA → Commander ($997)                         [ACHAT]
  CTA → Commander ($1597)                        [ACHAT]
  CTA → "Se faire rappeler"                      [CONVERSION]

/visa-e2/ :
  CTA → "Cliquez ici pour être recontacté"       [CONVERSION]

/creer-llc-offre/ :
  CTA → /contact                                 [CONVERSION]

FOOTER (toutes pages) :
  CTA → Calendly (entretien découverte)          [PERMANENT]
```

### Tunnel principal (parcours idéal)
```
ENTRÉE (Google/bouche-à-oreille)
    ↓
HOMEPAGE
    ├── Option A : CTA direct → Calendly (fast track)
    ├── Option B : Formation gratuite → capture email → nurture → Calendly
    └── Option C : Explorer → /llc-usa/ (pricing) → Commander/Se faire rappeler
                        ou → /creer-llc-offre/ (détail) → /contact → Calendly
                        ou → /visa-e2/ → être recontacté
                        ou → FAQ → rassurance → retour homepage → Calendly
```

### Points de friction identifiés
- ❌ Pas de sticky bar → le CTA disparaît quand on scroll
- ❌ Pas de pricing sur la homepage → faut aller sur /llc-usa/
- ❌ Le lien "Commander" sur /llc-usa/ → mène où ? (pas clair)
- ❌ /creer-llc-offre/ et /llc-usa/ contiennent du contenu similaire → confusion
- ❌ La FAQ est sur une page séparée → le prospect doit quitter le flow
- ❌ "Se faire rappeler" → pas de formulaire visible, juste un lien
- ❌ Le blog n'est pas lié au funnel (pas de CTA dans les articles)

### Convergence des pages
```
                    ┌── /creer-llc-offre/ ──┐
                    ├── /llc-usa/ ──────────┤
HOMEPAGE ──→        ├── /visa-e2/ ──────────┼──→ Calendly (RDV)
                    ├── FAQ ────────────────┤         ou
                    ├── /blog/ ─────────────┤    Formation gratuite
                    └── /contact/ ──────────┘
```
Toutes les pages convergent vers 2 destinations : **Calendly** ou **Formation gratuite**.
Mais la convergence est lente — beaucoup de pages intermédiaires avant d'arriver au CTA.

## 2. PARTNERSLLC.FR

### Sitemap (observable depuis la homepage)
```
/                           Homepage (LP unique, très longue)
└── EXTERNES :
    ├── Calendly ou équivalent (RDV 15 min)
    └── Portail client (Connexion)
```
C'est une **single-page LP**. Pas de sous-pages visibles. Tout est sur la homepage.

### Sections de la homepage (le funnel EST la page)
```
1. NAV          → Connexion + Prendre RDV
2. TRUSTPILOT   → 4.7/5, 27 avis
3. AVATARS      → Photos clients
4. HERO         → Titre + sous-titre + disclaimer
5. META         → "Durée 4 min · Pour e-com, agence, freelance"
6. VIDEO        → Video embed (Loom/VSL)
7. STICKY BAR   → "430€ · 48h · Sécurisé · Garantie 30j · RDV"
8. RDV SECTION  → "En 15 min, recommandation claire"
9. PROCESS      → "Comment ça marche en 4 étapes"
10. TESTIMONIALS VIDEO → Vidéos clients
11. PRICING     → Offres (packs)
12. PLATEFORME  → "La plateforme n°1 francophone"
13. SANS/AVEC   → Comparaison avant/après LLC
14. TRUSTPILOT  → Avis détaillés
15. FAQ         → Questions fréquentes
```

### Points de conversion
```
STICKY BAR (permanent, toutes sections) :
  CTA → Réserver RDV (15 min)                   [PRIMAIRE - PERMANENT]

NAV :
  CTA → Prendre RDV                             [PRIMAIRE]
  CTA → Connexion                               [RÉTENTION]

SECTION RDV :
  CTA → Réserver                                 [PRIMAIRE]

PRICING :
  CTA → Commander / Choisir l'offre              [ACHAT]
```

### Tunnel principal
```
ENTRÉE (Ads Meta/Google)
    ↓
HOMEPAGE (single page)
    ↓ scroll continu
Trustpilot → Video VSL → Sticky bar omniprésente
    ↓
Le prospect est éduqué + rassuré pendant le scroll
    ↓
À N'IMPORTE QUEL MOMENT → Sticky bar → RDV (15 min)
    ou
PRICING (milieu de page) → Commander
    ou
FAQ (bas de page) → dernière objection levée → RDV
```

### Points de friction
- ✅ Quasi zéro friction — tout est sur une page
- ✅ Sticky bar = CTA permanent, impossible à rater
- ❌ Pas de lead magnet (pas de formation gratuite, pas de capture email)
- ❌ Pas de contenu profond (pas de blog, pas de FAQ juridique détaillée)
- ❌ Le prospect qui n'est pas prêt n'a aucun chemin alternatif

### Convergence
```
TOUTE LA PAGE ──→ 1 seul point : RDV (Calendly ou équivalent)
```
C'est un **entonnoir pur**. Pas de distraction, pas de bifurcation.

```
NAV (toutes pages) :
  CTA → Request a Demo                          [PRIMAIRE]
  CTA → Login                                   [RÉTENTION]

HOMEPAGE :
  CTA → Request a Demo (hero)                   [PRIMAIRE]
  CTA → Our Customers (logo bar)                [EXPLORATION]

/customers/ :
  CTA → Request a Demo                          [PRIMAIRE]

  CTA → Request a Demo                          [PRIMAIRE]
```

### Tunnel principal
```
ENTRÉE (referral, recherche, bouche-à-oreille enterprise)
    ↓
HOMEPAGE → "Practice Made Perfect"
    ├── Option A : Request a Demo (fast track) → formulaire enterprise
    └── Option B : Explorer →
        ├── /platform/ → comprendre le produit → Request a Demo
        ├── /solutions/ → trouver son use case → Request a Demo
        ├── /customers/ → voir la social proof → Request a Demo
        └── /security/ → vérifier la compliance → Request a Demo
```

### Points de friction
- ❌ "Request a Demo" = formulaire enterprise, pas un booking direct
- ❌ Pas de prix visible nulle part → impossible de self-serve
- ❌ Pas de lead magnet, pas de contenu gratuit actionnable
- ❌ Long parcours avant le CTA (beaucoup de pages d'exploration)
- ✅ Mais chaque page converge vers le même CTA
- ✅ La social proof est massive (Fortune 500 logos, stats, vidéos)

### Convergence
```
TOUTES LES PAGES ──→ 1 seul point : Request a Demo (formulaire)
```
C'est un **entonnoir enterprise** — lent mais inexorable. Chaque page ajoute de la confiance.

## COMPARAISON DES TUNNELS

| **Type** | Multi-pages dispersé | Single-page LP | Multi-pages convergent |
| **Nb de pages** | 10+ | 1 | 8+ |

| **Sticky CTA** | ❌ Non | ✅ Oui | ❌ Non |

| **FAQ** | ✅ Page dédiée | ✅ Sur la LP | ❌ Non |

| **Profondeur** | Haute | Basse | Haute |

## CE QU'ON DOIT CONSTRUIRE

Le tunnel idéal combine :
- **La convergence de Partners LLC** : 1 CTA principal, sticky, impossible à rater
- **La profondeur de MA Societe US** : FAQ, blog, Visa E2, contenu éducatif
- **L'autorité de Harvey.ai** : chaque page renforce la confiance
- **Le lead magnet de MA Societe US** : formation gratuite comme chemin alternatif

```
TUNNEL CIBLE :

ENTRÉE (Ads / SEO / bouche-à-oreille)
    ↓
HOMEPAGE (LP structurée, pas single-page)
    ├── FAST TRACK : Sticky bar → Calendly (0 clic)
    ├── ÉDUCATION : VSL ou video embed → Calendly (1 clic)
    ├── LEAD MAGNET : Formation gratuite → email capture → nurture → Calendly
    └── EXPLORATION :
        ├── /creer-llc/ → carte interactive → /creer-llc/[état] → Calendly
        ├── /services/pack-llc/ → pricing détaillé → Calendly
        ├── /immigration-us/ → Visa E2 → Calendly
        └── /blog/ → article → CTA inline → Calendly
            
CHAQUE PAGE → sticky bar + CTA section → Calendly
AUCUNE PAGE ne se termine sans un CTA vers Calendly ou Formation gratuite
```
