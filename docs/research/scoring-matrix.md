# Matrice de scoring — Nouveau site MA SOCIETE US

## Source : instructions Benjamin Koch (avril 2026)

4 piliers : **Confiance, Compétence, Simplicité, Conviction**
+ contraintes : Mobile-first, DR style, autorité institutionnelle, rien au hasard

---

## PILIER 1 — CONFIANCE

"Le prospect confie sa structure juridique et fiscale. C'est pas un achat à 29€."

| Critère | Ce qu'on évalue | Score 1-5 |
|---|---|---|
| **Trust signals** | Trustpilot, avis, étoiles, nombre de clients | |
| **Social proof** | Témoignages avec noms, photos, spécificités | |
| **Crédibilité visuelle** | Le site "fait sérieux" — pas cheap, pas template | |
| **Pricing transparent** | Prix visibles, pas de surprises, pas de "sur devis" | |
| **Légitimité** | Adresse, téléphone, noms réels, mentions légales | |
| **Rassurance** | Section FAQ, section "est-ce légal", disclaimers | |

## PILIER 2 — COMPÉTENCE

"Se démarquer tout en conservant une forte autorité institutionnelle."

| Critère | Ce qu'on évalue | Score 1-5 |
|---|---|---|
| **Expertise affichée** | Années d'expérience, nombre de sociétés créées, équipe | |
| **Spécificité** | Contenu détaillé par état, par service — pas du générique | |
| **Qualité du contenu** | Les textes montrent une vraie connaissance (fiscalité, droit) | |
| **Design authority** | Le craft visuel communique "on est les meilleurs" | |
| **Interactivité** | Carte interactive, calculateur, outil — pas juste du texte | |
| **Blog/ressources** | Articles de fond, formation, contenu éducatif | |

## PILIER 3 — SIMPLICITÉ

"Objectivement moins clair et convergent que leur site initial sur mobile."

| Critère | Ce qu'on évalue | Score 1-5 |
|---|---|---|
| **Clarté du message** | En 5 secondes, on comprend ce que fait le site | |
| **Hiérarchie visuelle** | L'oeil sait où aller — titre → sous-titre → CTA | |
| **Navigation** | Peu d'items, logique, pas de confusion | |
| **Mobile** | Lisible, pas de scroll horizontal, CTAs accessibles au pouce | |
| **Densité** | Pas trop de texte, pas trop vide — juste ce qu'il faut | |
| **Flow** | Le scroll raconte une histoire linéaire sans détour | |

## PILIER 4 — CONVICTION

"Le site vitrine doit inspirer surtout conviction. Agressivité opérationnel."

| Critère | Ce qu'on évalue | Score 1-5 |
|---|---|---|
| **CTA clarity** | Le prospect sait quoi faire à chaque instant | |
| **CTA frequency** | Un CTA toutes les 2-3 sections (pas juste en haut et en bas) | |
| **Urgence** | Éléments de rareté/délai (24h, places limitées, countdown) | |
| **Objection handling** | FAQ, "est-ce légal", "pourquoi nous", avant/après | |
| **Bénéfice concret** | Chiffres : "30K€ économisés", "0% d'impôts", "$997" | |
| **Lead capture** | Formation gratuite, email capture, funnel secondaire | |

---

## SCORING

Chaque critère : 1 (absent/mauvais) → 5 (excellent/best-in-class)
Total par pilier : /30
Total global : /120

### Seuils
- 100+/120 : Best-in-class, prêt à pitcher
- 80-99 : Bon, quelques trous à combler
- 60-79 : Moyen, des piliers faibles
- <60 : À refaire
