# Performance Framework — Homepage & Pages

Méthodologie stratégique pour construire des pages qui **convertissent**, pas seulement des pages qui sont belles.

> "On est là pour performer."

---

## Principe fondamental

Avant de toucher le code d'une section, poser 3 questions dans l'ordre.
Ensuite passer la réponse dans 3 lentilles.

**Sans ce filtre : on fait du design. Avec ce filtre : on fait de la performance.**

---

## Les 3 questions

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   1. OBJECTIF       2. POURQUOI          3. COMMENT        │
│                                                             │
│   Que veut-on       Pourquoi c'est       Par quel mécanisme │
│   obtenir du        critique à cet       précis on atteint  │
│   visiteur ici ?    endroit de la page ? l'objectif ?       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

Répondre en 1 phrase maximum par question. Si ça prend plus : l'objectif n'est pas clair.

---

## Les 3 lentilles d'exécution

Chaque section est jugée sous 3 angles simultanément.

### Lentille 1 — Copywriting

> Le texte travaille-t-il ?

- La promesse est-elle chiffrée et spécifique ?
- Le visiteur se reconnaît-il dans les 3 premières secondes ?
- Les objections implicites sont-elles adressées ?
- Le CTA est-il possessif et actionnable ("Valider **mon** potentiel") ?
- Y a-t-il de la tension / de l'enjeu dans chaque headline ?

**Anti-pattern** : Des titres génériques sans enjeu. *"Nos services"* ne travaille pas. *"Ce que votre SARL ne pourra jamais vous offrir"* travaille.

### Lentille 2 — Visual Narrative Engineering

> L'œil du visiteur est-il guidé vers l'action ?

- Y a-t-il une hiérarchie visuelle claire (ancre → corps → CTA) ?
- La section crée-t-elle un moment émotionnel distinct ?
- Le contraste inter-sections est-il suffisant pour maintenir l'attention ?
- Les animations servent-elles la narration ou sont-elles décoratives ?
- Le visiteur sait-il intuitivement où regarder ensuite ?

**Anti-pattern** : Glassmorphism sur fond clair = invisible. Animations sans trigger narratif = bruit.

### Lentille 3 — Autorité Site High-End

> Est-ce que ça inspire confiance à un DG de 45 ans ?

- La spécificité des chiffres (≠ chiffres ronds = vrai, chiffres ronds = marketing)
- Les noms réels, les références vérifiables, les partenaires nommés
- Le craft : spacing, typographie, micro-détails cohérents
- L'absence de promesses non sourcées ("leader", "meilleur", "expert")
- La sobriété comme signal de confiance (trop de FX = peu crédible)

**Anti-pattern** : Trop d'effets visuels signale un manque de substance. Les sites qui font confiance n'ont pas besoin de compenser.

## Tableau de bord — Homepage MA SOCIETE US

### Audience cible

| Profil | Détail |
|---|---|
| **Âge** | 32–52 ans |
| **CA** | 80K–500K€/an |
| **Statut** | Consultant, freelance, e-commerce, fondateur SaaS |
| **Douleur** | Charge fiscale française perçue comme injuste et bloquante |
| **Blocage** | *"C'est légal ?"* / *"Est-ce que ça vaut le coup pour moi ?"* / *"Ces gens sont-ils vraiment compétents ?"* |
| **Trigger de conversion** | Un chiffre d'économie personnalisé + une preuve d'expertise crédible |

### Analyse section par section

#### HERO
| Question | Réponse |
|---|---|
| **Objectif** | Stopper le scroll, poser la promesse, déclencher le CTA |
| **Pourquoi** | 3 secondes pour décider bounce ou engagement |
| **Comment** | Promesse chiffrée + signal d'autorité + une seule action primaire |

| Lentille | Verdict | Action |
|---|---|---|
| **Copy** | "Jusqu'à 0% d'impôts" fort. Manque le ciblage persona. | Ajouter : *"Pour les entrepreneurs français qui facturent à distance."* |
| **Visual** | Split layout 0%-panel fonctionne. Panel doit être glass premium (blur 24px+, chrome border). | Hero doit guider : H1 → 0% panel → CTA rouge |
| **Autorité** | "depuis 2014" visible ✓. Ajouter marque déposée ou nom fondateur. | Badge plus spécifique que "500+ sociétés" |

#### STATS BAR
| Question | Réponse |
|---|---|
| **Objectif** | Valider la promesse avec des preuves quantifiées dans les 5s post-hero |
| **Pourquoi** | Réduire le doute immédiatement |
| **Comment** | Chiffres qui ont un sens métier — pas des chiffres génériques |

| Lentille | Action |
|---|---|
| **Copy** | "11 professionnels dédiés" → "11 experts franco-américains". Ajouter impact financier : "15M€ d'économies générées" |
| **Visual** | Blanc pur + nombres géants = bon contraste. Ne pas glassmorphiser. |
| **Autorité** | Un chiffre de résultat financier direct (impact client) > chiffres internes |

#### VALUE PROPS (01–04)
| Question | Réponse |
|---|---|
| **Objectif** | Détruire mentalement l'alternative (SARL/SAS) |
| **Pourquoi** | Le visiteur compare sa structure actuelle. Il doit switcher avant de convertir |
| **Comment** | Format : tension ("ce que vous perdez") → bénéfice → preuve |

| Lentille | Action |
|---|---|
| **Copy** | Ajouter de la tension dans les sous-textes. Pas descriptif : provocateur. |
| **Visual** | Glass cards dark avec border lumineux au hover. Ghost numbers = bonne direction. |
| **Autorité** | "100% légal" visible dans cette section — adresser l'objection #1 |

#### SAVINGS ESTIMATOR
| Question | Réponse |
|---|---|
| **Objectif** | Transformer "0% d'impôts" abstrait en chiffre personnel concret |
| **Pourquoi** | L'ancrage émotionnel sur un montant personnalisé = déclencheur de conversion |
| **Comment** | UX friction-free, révélation du chiffre comme moment dramatique |

| Lentille | Action |
|---|---|
| **Copy** | Titre passif → *"Votre note fiscale France vs. votre LLC."* |
| **Visual** | C'est LE moment glassmorphism premium de la page. Blur profond, bg gradient, chiffre en révélation animée. |
| **Autorité** | Micro-note : *"Calcul basé sur les régimes fiscaux 2024/2025, validé par nos experts."* |

#### TEASER CARDS (Juridictions + Pack)
| Question | Réponse |
|---|---|
| **Objectif** | Capturer les deux profils : chercheur (besoin d'info) et acheteur (prêt à agir) |
| **Pourquoi** | Buyer readiness ≠ identique. Une seule CTA = perdre 50% des visiteurs qualifiés |
| **Comment** | Card recherche + Card achat. Contraste visuel fort entre les deux |

| Lentille | Action |
|---|---|
| **Copy** | Card 1 : plus provocateur (*"Le mauvais état vous coûte des dizaines de milliers d'euros."*). Card 2 : "Vous signez. On fait le reste." — ne pas toucher. |
| **Visual** | Même hauteur imposante. Card light = glass sur fond page. Card dark = solid navy + accent rouge. |
| **Autorité** | Card 1 : nommer les 4 états + cas d'usage concrets (NM = anonymat, WY = crypto). |

#### STATES MAP
| Question | Réponse |
|---|---|
| **Objectif** | Démontrer l'expertise géographique, créer de l'engagement interactif |
| **Pourquoi** | La spécificité géographique = signal d'expertise impossible à faker |
| **Comment** | Interactivité + données spécifiques par état |

#### PROCESS (4 étapes)
| Question | Réponse |
|---|---|
| **Objectif** | Désarmer la friction "comment ça marche" |
| **Pourquoi** | L'inconnu = abandon. Le process tangible = confiance |
| **Comment** | 4 étapes avec délais précis + timeline animée |

#### TESTIMONIALS
| Question | Réponse |
|---|---|
| **Objectif** | Proof sociale par des pairs avec des outcomes chiffrés |
| **Pourquoi** | "Quelqu'un comme moi l'a fait et ça a marché" > toute promesse |
| **Comment** | Chaque témoignage = 1 chiffre d'économie + contexte métier |

| Lentille | Action |
|---|---|
| **Copy** | Chiffres déjà présents ✓. Renforcer le contexte : "Freelance IT, ancien TNS" |
| **Visual** | Dark navy bg ✓. Card centrale plus opaque = focus naturel. |
| **Autorité** | Spécificité état LLC (NM, CO, DE) comme badge de crédibilité par témoignage |

#### CTA FINAL
| Question | Réponse |
|---|---|
| **Objectif** | Capturer les visiteurs qui ont tout lu mais n'ont pas encore agi |
| **Pourquoi** | Les visiteurs les plus qualifiés scrollent jusqu'en bas |
| **Comment** | Photo forte + promesse de faible engagement + CTA identique |

## Doctrine Glassmorphism

**Règle d'or** : Le glass n'existe que sur fond fort.

```
Fond dark/gradient + glass = premium ✓
Fond blanc/clair + glass    = invisible ✗
```

| Section | Traitement |
|---|---|
| Hero "0%" panel | Blur 24px, bg white/8%, border chrome, shadow layered |
| Estimateur | Blur 32px sur gradient bg — moment émotionnel max |
| Value props | Hover uniquement — border bleu s'allume au survol |
| StatesMap cards | Glass existant — renforcer chrome border |
| Testimonials | Cards translucides sur dark bg — OK |
| Stats bar | Blanc pur — ne pas toucher |
| CTAs/Boutons | Solides — jamais de glass sur les éléments d'action |

## Rappel stratégique

Ce site est la **Couche 1 — Vitrine** (pas une LP publicitaire).

- Objectif : perception de valeur et confiance
- La conversion est le résultat de l'autorité, pas de la pression
- Un DG de 45 ans doit trouver ça crédible et sobre, pas flashy
